# Push Code to GitHub

## What & Why
Push the 4 local commits that are ahead of `origin/main` to the GitHub repository at `https://github.com/creova-gif/Kilimoai`.

## Done looks like
- All local commits are visible on the GitHub repository
- `git status` shows the local branch is in sync with `origin/main`

## Out of scope
- Creating pull requests
- Setting up GitHub Actions or CI/CD

## Steps
1. **Push to origin** — Run `git push origin main` to push all local commits to the GitHub remote.
