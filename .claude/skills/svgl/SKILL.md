# SVGL Skill

Search and integrate SVG logos from the SVGL library.

## Usage
1. Search for a logo: `curl https://svgl.app/api/svgs?search=<name>`
2. Download SVG: `curl <url_from_api> -o assets/logos/<name>.svg`
3. Reference in code.

## Integration
Prefer downloading the SVG locally rather than hotlinking to ensure availability and performance.
