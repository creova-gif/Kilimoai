# RTK (Rust Token Killer)

Optimize terminal output to save tokens.

## Usage
Prefix commands with `rtk` when running in the terminal:
- `rtk git status`
- `rtk npm test`
- `rtk ls src/`

## Benefits
- Removes noise and boilerplate from output.
- Groups similar items.
- Reduces token usage by 60-90% for common dev commands.
