## 🚀 **Getting Started**

### **For Farmers:**
1. Sign up → Choose "Smallholder Farmer"
2. Complete AI-guided registration
3. Verify phone number (OTP)
4. Explore dashboard quick actions
5. Ask Sankofa AI your first question

### **For NGOs/Extension Officers:**
1. Sign up → Choose "Extension Officer"
2. Set up organization profile
3. Add farmers to network
4. Access analytics dashboard
5. Schedule field visits

### **For Cooperatives:**
1. Sign up → Choose "Cooperative Leader"
2. Set up cooperative details
3. Add member farmers
4. Plan harvest aggregation
5. Coordinate group sales
