## 🆘 Troubleshooting

### Farm Graph not saving data
- Check internet connection
- Try again (auto-saves to localStorage first)
- Contact Support if persists

### Voice Assistant not recognizing Swahili
- Speak clearly near microphone
- Reduce background noise
- Try text fallback option

### AI Recommendations seem generic
- Complete more Farm Graph entries
- Update your crop preferences
- Give feedback on recommendations

### Farmer Lab application stuck
- Check eligibility criteria
- Ensure profile is complete
- Contact Support for status update

### Family Planner privacy concerns
- Use privacy controls per member
- Only share with trusted household members
- Can delete data anytime

### Image Diagnostics low confidence
- Take multiple angles
- Ensure good lighting
- Upload clearer image
- Consult extension officer
