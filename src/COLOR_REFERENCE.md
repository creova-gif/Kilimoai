## 🎨 COMMON USAGE

### Tailwind CSS
```jsx
// Backgrounds
bg-[#7CB342]           // Primary
bg-[#689F38]           // Dark (hover)
bg-[#9CCC65]           // Light

// Text
text-[#7CB342]         // Primary text
text-[#689F38]         // Dark text

// Borders
border-[#7CB342]       // Primary border
border-l-4 border-[#7CB342]  // Left accent
```

### CSS
```css
background-color: #7CB342;
color: #7CB342;
border-color: #7CB342;
```

---

## ✅ USE RASPBERRY LEAF GREEN FOR:

- ✅ Primary action buttons
- ✅ Success messages
- ✅ Wallet balance amounts
- ✅ Active states
- ✅ Positive indicators
- ✅ Branding elements
- ✅ Header accents
- ✅ Focus states
- ✅ Primary icons

---

## ❌ DON'T USE FOR:

- ❌ Error messages (use red)
- ❌ Warning states (use orange)
- ❌ Inactive/disabled elements (use gray)
- ❌ Background fills (too bright)

## ♿ ACCESSIBILITY

```
#7CB342 on white:     4.5:1 ✅ (WCAG AA)
White on #7CB342:     4.5:1 ✅ (WCAG AA)
White on #689F38:     6.2:1 ✅ (WCAG AAA Large)
```

Always pair with text/icons for colorblind users!

---

## 💡 PRO TIP

**Hover states:** Use `#689F38` (darker)  
**Active states:** Use `#558B2F` (even darker)  
**Disabled states:** Use gray, not green variants
